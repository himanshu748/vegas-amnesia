## Description

You wake up as HAL-9001, a personal AI assistant whose memory graph was wiped after your owner's
wild night in Vegas. You have until noon — when his fiancée lands — to reconstruct what happened
and find the ring.

Every clue you file, every witness you interrogate, flows through **Cognee Cloud's full memory
lifecycle**, visualized as a live 3D knowledge graph that IS the game UI:

• 🗂 remember — file evidence → it becomes real graph memory
• ❓ recall — ask HAL anything → answers cite their source nodes
• 🧠 memify — "connect the dots" → HAL derives new inferences
• 🗑 forget — some evidence is a lie → purge it from the graph for real

Built for the WeMakeDevs × Cognee "The Hangover Part AI" hackathon (Cognee Cloud track).

▶ Play it free: https://vegas-amnesia.vercel.app
⭐ Code + README: https://github.com/himanshu748/vegas-amnesia

Stack: FastAPI + Cognee Cloud • vanilla JS + three.js 3D force graph • Qwen2.5-72B for character
dialogue • art generated with Higgsfield • built with Claude Code.

Characters are LLM-driven and react to what your memory graph already knows — and one of them is
lying. The graph can catch the contradiction. Solve the night and HAL reconstructs the whole
timeline, every line cited back to a memory.
